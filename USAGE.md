# LivePuzzle 使用说明

## 编辑器功能

### 🎨 自由画布体验
- **无限画布**：图片显示在灰色画布上，可以自由缩放和拖动
- **移除背景**：图片单元格没有白色背景，更清爽
- **自由定位**：可以将拼图缩小到极小，在整个画布上自由移动

### 🔍 缩放功能
- **双指捏合**：在编辑区域使用双指捏合手势可以放大/缩小拼图
- **缩放范围**：0.01x - 10.0x（几乎无限制缩小，最大放大到1000%）
- **无限拖动**：缩放后可以在无限大的画布上自由拖动
- **极度缩小**：可以将整个长拼图缩小到一个点，便于全局定位
- **超级放大**：可以放大到 10 倍查看每个像素的细节

### 🎯 选择/取消选择图片
- **选中图片**：点击拼图中的任意图片单元格
- **取消选中**：点击拼图边缘的灰色背景区域
- **指示**：选中的图片会显示粉色边框（3px）
- **帧选择器**：只在选中图片时显示

### 🎬 设置封面
1. 点击选中要设置封面的图片
2. 底部会显示视频播放器和进度条
3. 拖动进度条预览不同时刻的画面
4. 点击"设置为封面"按钮保存当前帧
5. 可以点击"重新设置封面"来更换封面

### ▶️ 播放 Live Photo
1. 点击顶部的"LIVE"按钮
2. 应用会自动提取所有视频帧（首次需要等待）
3. 拼图会像 Live Photo 一样播放动画
4. 再次点击可以暂停

### 💾 保存
1. 点击右上角的"完成"按钮
2. 应用会生成 Live Photo 格式的拼图
3. 保存到相册后，可以在照片应用中查看
4. 支持实况定格功能

## 技术细节

### 元数据
保存的 Live Photo 包含以下元数据：
- `com.apple.quicktime.content.identifier`：Live Photo 配对标识符
- MakerApple Dictionary `17`：图片视频配对 ID
- MakerApple Dictionary `8`：Still Image Time（支持实况定格）

### 视频规格
- 帧率：15 FPS
- 帧数：30 帧
- 时长：2 秒
- 编码：H.264
- 分辨率：根据图片自动调整（最大 1200px 宽度）
